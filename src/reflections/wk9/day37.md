# DAY 037

## What is a Github action and how do they work?

Its built in a YAML file and they are actions that you want github to run when certain listeners are triggered.

## What benefits do Github actions provide?

They can help mitigate any unwanted forks or pull requests. 

## What types of trigger actions can a workflow use? What do they do?

A fork, a pull request or a merge could all be used to trigger a specific action through github. The will do whatever you ask them to do.
